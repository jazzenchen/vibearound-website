## Release Highlights

Version 0.7.1 promotes the 0.7.1 line from beta to stable. It restores image/file capability metadata through the API bridge, clarifies desktop launch routing, improves profile visibility in runtime status, fixes stale IM attachment records, aligns Startkit toolchain defaults, and restores Windows MSI availability.
