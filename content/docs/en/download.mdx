## Release Highlights

Version 0.7.16 keeps rapid follow-up messages and streaming assistant output in chronological order in the web dashboard.

The command-line distribution is published separately as [VibeAround CLI 0.0.6](https://github.com/jazzenchen/VibeAround/releases/tag/va-v0.0.6). Use the desktop links above for the packaged app, and install the current CLI from npm with:

```bash
npm i -g @vibearound/cli
```
