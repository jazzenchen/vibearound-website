## Release Highlights

Version 0.7.10 is an architectural release. It separates the local server, launch layer, desktop UI, Web Hub, CLI, and early TUI so they can run and evolve more independently. It adds `va serve`, introduces standalone `va-launch`, and makes the command-line distribution a beta path for terminal-first workflows.

This release changes how local configuration is read and normalized. Back up `~/.vibearound/profiles` and `~/.vibearound/settings.json` before upgrading from older versions, then follow the migration notes in the GitHub release if API Bridge settings need to be re-enabled.
