## Release Highlights

Version 0.7.13 adds profile-scoped API configuration, updates the ACP bridge/runtime stack, and improves web chat, mobile pairing, local bridge, and route recovery reliability.

The command-line distribution is published separately as [VibeAround CLI 0.0.3](https://github.com/jazzenchen/VibeAround/releases/tag/va-v0.0.3). Use the desktop links above for the packaged app, and install the current CLI from npm with:

```bash
npm i -g @vibearound/cli
```
