## Release Highlights

Version 0.7.3 unifies agent executable detection across Startkit, install checks, app launch, and launch-agent usage. It improves CLI launch path selection so detected candidates keep source and version metadata when saved as defaults, shows lower-ranked installs instead of hiding them, separates launch settings from path selection, localizes launch path dialogs, and clarifies desktop app targets across macOS and Windows. It also fixes save-time detection churn, launch path dialog layout and scroll behavior, candidate update actions, latest-version indicators, and platform-aware path validation.
