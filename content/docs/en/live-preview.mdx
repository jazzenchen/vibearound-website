---
title: Live Preview for AI Coding Agents
description: Preview local dev servers, Markdown, HTML, and AI-generated artifacts from desktop, browser, mobile, or messaging surfaces.
---

Share what is running on your machine without deploying:

- **Dev server previews.** Register a local port and get a preview page that reverse-proxies it, with an iframe toolbar. Agents create these automatically when they start dev servers (via the `va-preview` skill / MCP `preview` tool — [tool reference](https://github.com/jazzenchen/VibeAround/blob/main/docs/reference/api-surfaces.md#mcp-tools)).
- **Markdown preview.** Any markdown file rendered GitHub-style (`md_preview` tool or the `va-md-preview` skill).
- **Two links per preview:** the owner URL (token-authenticated, lives as long as the preview) and a share URL that expires after 10 minutes and needs no auth — safe to paste in a group chat. See [Security model](/docs/local-security).
- `va previews` / `va preview delete <slug>` manage them from the CLI; preview processes started for you are killed when the daemon stops.

## Preview URLs
| URL | Auth | Lifetime |
|---|---|---|
| `/preview/u/{slug}` | Owner token | While the preview exists |
| `/preview/s/{slug}` | None | 600 s |
| `/md-preview/{slug}` | Owner token | While it exists |
