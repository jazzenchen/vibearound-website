## 版本要点

v0.7.1 将 0.7.1 从 beta 提升为稳定版。该版本恢复了 API Bridge 中图片/文件能力 metadata，明确了桌面 Agent 的 Launch 路由展示，改进了 Runtime Status 中的 Profile 可见性，修复了旧 IM attach 记录残留的问题，统一了 Startkit toolchain 默认值，并恢复了 Windows MSI。
