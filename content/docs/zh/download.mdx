## 版本要点

v0.7.3 统一了 Startkit、安装检查、应用启动和 launch-agent 使用场景中的 Agent 可执行文件检测。该版本改进了 CLI 启动路径选择，让检测到的候选项在保存为默认路径时保留来源和版本信息；低优先级安装项也会继续展示，而不是被隐藏；启动设置和路径选择被拆分，路径选择弹窗增加了本地化，并进一步统一了 Codex、Claude CLI 和桌面端路径 UI。该版本还修复了保存时重复检测、启动按钮短暂收缩、路径弹窗布局和滚动、候选项更新操作、最新版标识以及跨平台路径校验问题。
