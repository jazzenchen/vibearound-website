## 版本要点

v0.7.15 收敛了进程、thread、channel、权限和路由的生命周期所有权，让 stop、cancel、restart、shutdown 和多路由会话行为更确定。

命令行发行包单独发布为 [VibeAround CLI 0.0.5](https://github.com/jazzenchen/VibeAround/releases/tag/va-v0.0.5)。需要桌面应用时用上面的安装包；如果偏好 npm，可以这样安装当前 CLI：

```bash
npm i -g @vibearound/cli
```
