## 版本要点

v0.7.16 修复了 Web 控制台里连续追问与流式回复的时间顺序，回复会从对应的最新用户消息之后继续。

命令行发行包单独发布为 [VibeAround CLI 0.0.6](https://github.com/jazzenchen/VibeAround/releases/tag/va-v0.0.6)。需要桌面应用时用上面的安装包；如果偏好 npm，可以这样安装当前 CLI：

```bash
npm i -g @vibearound/cli
```
