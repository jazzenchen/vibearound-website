## 版本要点

v0.7.13 新增按 Profile 配置 API 的能力，并升级 ACP bridge/runtime 链路，同时改进 Web Chat 重连、移动端配对页、本地 bridge 安全和 route 恢复可靠性。

命令行发行包单独发布为 [VibeAround CLI 0.0.3](https://github.com/jazzenchen/VibeAround/releases/tag/va-v0.0.3)。需要桌面应用时用上面的安装包；如果偏好 npm，可以这样安装当前 CLI：

```bash
npm i -g @vibearound/cli
```
