---
title: Gemini CLI 远程访问
description: 从浏览器、手机、Web Terminal、消息频道和 handover 流程访问本地 Gemini CLI 会话。
---

VibeAround 可以让 Gemini CLI 会话在本地工作空间旁继续运行，同时从手机或浏览器访问它。适合希望远程触达 Gemini CLI，但不想把项目迁移到托管环境的开发者。

Gemini 是 Google 产品。VibeAround 是协调本地 Agent 工作流的独立软件。

## 设置路径

1. 在 VibeAround 外部安装并认证 Gemini CLI。
2. 确认它能访问目标工作空间。
3. 在 VibeAround 中加入该工作空间。
4. 通过 [Agent 启动](./agent-launch) 开始或继续会话。
5. 本地基线稳定后，再加入 handover、Web Terminal、消息频道或预览入口。

## 常见用途

- 当另一个 Agent 正在修改时，请 Gemini CLI 给出第二意见。
- 本地主机保持在线时，从手机继续调查。
- 工作流需要显式 endpoint 或模型选择时，使用 Provider Profile。
- 通过 VibeAround 预览入口审阅生成的 Markdown 或网页。

## 排障

如果 Gemini CLI 在 VibeAround 中不可用，先在同一工作空间里直接运行它。然后检查终端模式、profile 路由、环境变量和本地认证状态。
