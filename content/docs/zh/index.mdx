VibeAround 是一个本地优先的 AI 编程 Agent 工作空间。它帮助你用合适的模型 Profile 启动合适的 Agent，从桌面、浏览器、终端、手机或消息频道继续同一个会话，并预览 Agent 生成的结果，同时让真实的仓库、终端和执行环境留在自己的电脑上。

这份文档不是首页的延伸，而是产品手册。新用户可以按顺序完成安装和第一次会话；日常用户可以查找 Launch、Workspace、Handover、Preview 等工作流；配置和运维人员可以查看 Provider、Channel、安全模型和排障；维护者可以查看架构与本地开发说明。

## VibeAround 协调什么

VibeAround 不替代你的仓库、终端、Agent CLI 或模型 Provider。它在这些工具外面提供一个统一控制层：

- 从一个入口启动 Claude Code、Codex CLI、Gemini CLI、Pi、OpenCode、Claude Desktop、Codex Desktop 等 Agent。
- 在启动前选择 Workspace、Terminal、Model Profile、API endpoint 和历史 Session。
- 从桌面 App、Web Dashboard、Web Chat、Web Terminal、手机浏览器和消息频道继续同一个本地会话。
- 为本地 dev server、Markdown、HTML 和生成文件创建可预览链接。
- 默认让执行留在本机，并对 tunnel、browser pairing、token、channel 和 preview link 提供明确控制点。
