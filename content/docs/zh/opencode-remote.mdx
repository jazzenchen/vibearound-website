VibeAround 中的 OpenCode 远程访问是一种本地优先工作流。OpenCode 在主机上靠近仓库和工具运行，VibeAround 提供浏览器、手机、终端、消息频道和预览入口回到该会话。

## 设置路径

1. 确认 OpenCode 可从本地终端运行。
2. 将项目目录加入 VibeAround 工作空间。
3. 通过 [Agent 启动](./agent-launch) 启动或连接会话。
4. 使用 [会话接续](./session-handover) 从其他设备继续。
5. 直接本地会话稳定后，再加入消息频道。

## 适合场景

- 希望围绕多个编程 Agent 使用一个工作空间。
- 希望从手机或浏览器终端访问 OpenCode。
- 工作流依赖本地开发服务、本地包缓存或私有网络。
- 会话产物需要通过预览链接审阅网页、Markdown 或 HTML。

## 需要验证的限制

支持程度可能随 OpenCode 版本、终端模式、会话持久化和 profile 路由而变化。在重要仓库中使用前，请先用小任务验证启动、继续和 provider 路由。
