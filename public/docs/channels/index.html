channels

Use your daily IM as an interface for local coding agents.

Each channel needs its own credentials, bot setup, or login flow. Start with one channel, confirm messages work, then add more.

Recommended setup order

  1. Enable one local agent and confirm it works from the desktop app.
  2. Add one IM channel and verify that /help responds.
  3. Run a small task before allowing longer command execution.
  4. Add handover with /handover and /pickup.

Keep credentials scoped

Use dedicated bot credentials where possible. Avoid sharing channel tokens with unrelated tools, and rotate them if a channel is no longer needed.
