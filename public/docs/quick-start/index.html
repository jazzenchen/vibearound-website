quick start

Get from install to your first local agent session.

Use this path when you want VibeAround running on your computer and reachable from desktop, browser, or IM.

1. Install the latest release

  1. Open the latest GitHub release.
  2. Download the package for your operating system.
  3. Launch VibeAround and complete the desktop onboarding wizard.

2. Choose a workspace

Add the project folder where your local coding agent should run. VibeAround routes agent sessions to local workspaces, so pick the same repository you would normally open in a terminal.

3. Enable an agent

Start with one agent first. VibeAround supports local workflows around Claude Code, Codex CLI, Gemini CLI, Cursor CLI, Kiro CLI, Qwen Code, and OpenCode.

Some agents may need their own CLI installed or authenticated before VibeAround can launch them.

4. Open the web dashboard

The local dashboard is exposed on your machine at http://127.0.0.1:12358. Use it for browser access, terminal sessions, tmux workflows, and agent chat.

5. Send a first task

Ask the agent to inspect the repository, run tests, or explain the project. Keep the first task small so you can verify workspace, provider, and command execution are all connected correctly.
