Privacy Policy

Last updated: May 9, 2026

VibeAround is built around local-first AI coding workflows. The desktop app is intended to run agent sessions on your own machine, beside your repositories, command line tools, and provider credentials.

Website data

The website is hosted on Cloudflare. Cloudflare may process routine request metadata such as IP address, user agent, requested URL, and timestamps to deliver the site, secure it, and understand operational health. The website does not require an account.

Release downloads

Release files may be served from Cloudflare R2 and mirrored from GitHub Releases. Download requests can create standard server and CDN logs. GitHub may also process data when you visit GitHub release pages or download directly from GitHub.

Desktop app data

VibeAround runs local coding agents and channel integrations configured by the user. Provider API requests, IM channel traffic, and repository access depend on the agents, model providers, and plugins that you choose to configure. Review each provider and channel service's own privacy policy.

Contact

For privacy questions, open an issue on GitHub or use the community links in the project README.
