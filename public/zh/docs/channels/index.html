快速开始 会话接续 IM 渠道 Provider 安全

channels

把日常 IM 变成本地 Coding Agent 的入口。

每个渠道都需要自己的 bot 凭据、应用配置或登录流程。建议先接一个渠道，确认消息可用，再逐步增加。

支持的渠道

渠道常见配置输入命令方式
TelegramBot token/command
飞书 / LarkApp credentials/command
DiscordBot token/command
SlackBot token 和 app token/va 或 /vibearound
微信扫码登录/command
钉钉AppKey 和 secret/command
企业微信Bot ID 和 secret/command
QQ BotApp ID 和 token/command

推荐配置顺序

  1. 先启用一个本地 Agent，并确认桌面端可用。
  2. 添加一个 IM 渠道，确认 /help 有响应。
  3. 先运行一个小任务，再允许更长的命令执行。
  4. 用 /handover 和 /pickup 测试会话接续。
