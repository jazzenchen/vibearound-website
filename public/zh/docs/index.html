中文文档

让本地 AI Coding Agent 从桌面、浏览器、手机和 IM 保持可触达。

如果你想安装 VibeAround、接入 Agent、把会话接续到手机、配置 IM 渠道、切换 Provider、 打开浏览器终端、分享本地预览，或理解本地优先安全边界，从这里开始。

快速开始

安装应用、选择 workspace、启用 Agent，并打开本地 Web Dashboard。

会话接续

用 `/handover`、`/pickup` 和 `/switch` 在桌面、浏览器和 IM 之间接续会话。

IM 渠道

连接 Telegram、飞书/Lark、Discord、Slack、微信、钉钉、企业微信或 QQ Bot。

Provider Profile

切换 Agent、模型 Provider、OpenAI-compatible endpoint 和本地 bridge route。

本地优先安全

理解哪些数据留在本地，哪些会经过模型 Provider、IM 渠道或 tunnel。

远程编程

从浏览器或手机继续本地 Agent 任务，不必把代码搬到云端 workspace。

Codex 手机工作流

让 Codex CLI 会话从手机友好的浏览器、终端、handover 和 IM 继续可触达。

Codex Remote 对比

理解官方 Codex remote、Codex cloud、SSH host 和本地优先多 Agent 接入的差异。

Agent Remote 页面

面向 Claude Code、Gemini CLI、OpenCode 和其他本地优先 coding agent 的 remote access 长尾页面。

VibeAround 连接了什么

VibeAround 连接本地 coding-agent CLI、本地 workspace、浏览器终端、IM 渠道、预览链接和统一 bridge route。 它适合 Agent 跑在你自己电脑上，但你想从别的入口继续控制同一个本地会话的场景。
