快速开始

从安装到第一个本地 Agent 会话。

这条路径适合第一次使用 VibeAround，并希望从桌面、浏览器或 IM 访问本地 Agent 的用户。

1. 安装最新版本

  1. 打开 GitHub latest release。
  2. 下载适合你操作系统的安装包。
  3. 启动 VibeAround，并完成桌面端 onboarding。

2. 选择 workspace

添加你希望 Agent 运行的项目目录。VibeAround 会把 Agent session 路由到本地 workspace，所以这里应该选择你平时会在 terminal 里打开的仓库。

3. 启用一个 Agent

先从一个 Agent 开始。VibeAround 支持围绕 Claude Code、Codex CLI、Gemini CLI、Cursor CLI、Kiro CLI、Qwen Code 和 OpenCode 的本地工作流。

某些 Agent 需要先单独安装 CLI 或完成登录认证，VibeAround 才能启动它们。

4. 打开 Web Dashboard

本地 Dashboard 运行在你的机器上：http://127.0.0.1:12358。你可以用它访问浏览器、terminal、tmux 工作流和 Agent chat。

5. 发送第一个任务

可以先让 Agent 检查仓库、运行测试或解释项目。第一个任务建议小一点，方便确认 workspace、provider 和命令执行都已经连通。
