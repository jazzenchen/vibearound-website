handover

让同一个本地 Agent 会话在 terminal、浏览器和 IM 之间接续。

会话接续是 VibeAround 的核心远程编程工作流：Agent 仍然跑在本地，但你可以从另一个入口继续对话。

基本流程

  1. 在 VibeAround 里启动或继续一个 Agent session。
  2. 在当前会话里发送 /handover。
  3. 复制生成的 /pickup CODE 命令。
  4. 把命令粘贴到已经连接的 IM 渠道或浏览器会话里。
  5. 从同一个本地上下文继续任务。

常用命令

命令用途
/help显示当前渠道可用命令。
/new开始一个新对话。
/switch <agent>切换到另一个已启用 Agent，例如 claude、codex 或 gemini。
/profile <name>切换 provider 或 route profile。
/handover导出当前会话，供其他入口 pickup。
/pickup <code>接续一个已经 handover 的会话。
/close关闭当前对话。

Slack 的命令前缀

Slack 会拦截 slash command。请使用 /va 或 /vibearound 前缀，例如 /va help 或 /va switch claude。
