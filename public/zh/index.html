v0.6 工作空间循环

每天的 Agent 工作，在这里开始。

01

启动与接续

在合适的本地工作目录启动任意 Agent，接续之前的会话。

02

在浏览器里工作

切换 Agent、上传文件、检查输出，会话始终可见。

03

审查代码变更

渲染好的 diff 和产物视图，看清每个 Agent 改了什么。

04

终端与预览

打开 PTY/tmux 会话，分享带鉴权的 dev server 预览。

05

切换 Agent 与模型供应商

在 Agent 和已保存的配置之间切换，API Bridge 负责模型路由。

06

在手机上继续

通过 Web Chat 或消息应用交接并继续工作。
